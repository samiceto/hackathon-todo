## 🔍 Accessing the Application

### Backend API (FastAPI)

The backend service is running on port {{ .Values.backend.service.port }}.

{{- if eq .Values.backend.service.type "ClusterIP" }}

Access via port-forward:

  kubectl port-forward svc/{{ include "todo-app.fullname" . }}-backend {{ .Values.backend.service.port }}:{{ .Values.backend.service.port }}

Then visit: http://localhost:{{ .Values.backend.service.port }}
Health check: http://localhost:{{ .Values.backend.service.port }}/health

{{- else if eq .Values.backend.service.type "NodePort" }}

Access via NodePort:

  minikube service {{ include "todo-app.fullname" . }}-backend

{{- else if eq .Values.backend.service.type "LoadBalancer" }}

Get the external IP:

  kubectl get svc {{ include "todo-app.fullname" . }}-backend

{{- end }}

### Frontend Application (Next.js)

The frontend service is running on port {{ .Values.frontend.service.port }}.

{{- if eq .Values.frontend.service.type "ClusterIP" }}

Access via port-forward:

  kubectl port-forward svc/{{ include "todo-app.fullname" . }}-frontend {{ .Values.frontend.service.port }}:{{ .Values.frontend.service.port }}

Then visit: http://localhost:{{ .Values.frontend.service.port }}

{{- else if eq .Values.frontend.service.type "NodePort" }}

Access via NodePort:

  minikube service {{ include "todo-app.fullname" . }}-frontend

Or directly via Minikube IP:

  MINIKUBE_IP=$(minikube ip)
  echo "Frontend URL: http://$MINIKUBE_IP:{{ .Values.frontend.service.nodePort }}"

{{- else if eq .Values.frontend.service.type "LoadBalancer" }}

Get the external IP:

  kubectl get svc {{ include "todo-app.fullname" . }}-frontend

{{- end }}

## 📊 Monitoring & Logs

View backend logs:

  kubectl logs -f -l app.kubernetes.io/component=backend,app.kubernetes.io/instance={{ .Release.Name }}

View frontend logs:

  kubectl logs -f -l app.kubernetes.io/component=frontend,app.kubernetes.io/instance={{ .Release.Name }}

Check pod resource usage (requires metrics-server):

  kubectl top pods -l app.kubernetes.io/instance={{ .Release.Name }}

## 🔧 Configuration

View backend configuration:

  kubectl get configmap {{ include "todo-app.fullname" . }}-backend-config -o yaml

View backend secrets (values are base64 encoded):

  kubectl get secret {{ include "todo-app.fullname" . }}-backend-secrets -o yaml

## 🛠️ Helm Management

Upgrade the deployment:

  helm upgrade {{ .Release.Name }} ./helm/todo-app -f values-dev.yaml

Rollback to previous version:

  helm rollback {{ .Release.Name }}

View deployment history:

  helm history {{ .Release.Name }}

Uninstall the application:

  helm uninstall {{ .Release.Name }}
