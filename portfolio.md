# Hackathon Todo

**Tagline:** AI-powered todo app for managing tasks through a web dashboard, chatbot, and voice input.

## Description

Hackathon Todo is a full-stack task management platform built with Next.js, FastAPI, PostgreSQL, and OpenAI. It supports secure user accounts, advanced task organization, AI chat-based task control, multilingual UI, voice input, reminders, and cloud-ready deployment with Docker, Kubernetes, Helm, Dapr, Redis, and Kafka/Redpanda.

## Features

- User authentication and private task lists.
- Create, update, complete, delete, search, filter, and sort tasks.
- Priorities, due dates, recurring tasks, reminders, and tags.
- AI chatbot for natural-language task actions.
- Voice input for speaking chatbot commands.
- English and Urdu support with RTL layout.
- Event-driven reminder service and Kubernetes-ready deployment.

## Chatbot Capacity

The chatbot can add tasks, list tasks, update details, mark tasks complete, delete tasks, remember conversation context, stream responses in real time, and ask helpful follow-up questions when a request is unclear.  
With voice chat, users can complete these actions by voice command in English or Urdu.

## Tech Stack

Next.js, React, TypeScript, Tailwind CSS, FastAPI, Python, SQLModel, PostgreSQL/Neon, OpenAI API, SSE streaming, Docker, Kubernetes, Helm, Dapr, Redis, Kafka/Redpanda, pytest, and Playwright.

## AI Role

AI works as a productivity assistant that converts natural-language requests into real task actions through backend tools.

## How to Use

Sign in, manage tasks from the dashboard, or use the chatbot with commands like "add a task", "show my high priority tasks", or "mark task 2 complete".
